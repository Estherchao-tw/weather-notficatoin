# 自動化天氣通知
## 自動化的完成這個任務呢?

使用自動化測試/部署(CI/CD)工具:heroku
透過heroku (Paas)做雲端佈署，透過雲端主機自動幫忙執行

### Environment
- python 3.8.3
- heroku

### Feature Stucture
- Web Scraping weather API
- post Line notify notification
- deploy to Heroku cloud Platform
- Heroku Schedule auto execute
